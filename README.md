# Meu App - O Senhor dos Anéis

Este é um projeto inicial em React Native utilizando o framework Expo. O aplicativo consiste em uma interface simples que exibe informações sobre a franquia de filmes **O Senhor dos Anéis**, utilizando componentes visuais do **React Native Paper**.

## 🚀 Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Native Paper](https://callstack.github.io/react-native-paper/)

## 📱 Funcionalidades

- Exibição de um Card com imagem e informações do filme "O Senhor dos Anéis".
- Layout responsivo utilizando `ScrollView`.
- Utilização de componentes estilizados do `react-native-paper` (`Card`, `Text`, `PaperProvider`).

## 🛠️ Como executar o projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. Clone o repositório ou baixe o projeto.
2. Acesse a pasta do projeto pelo terminal:
   ```bash
   cd "Projeto Inicial Mobile/meu-app"
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor do Expo:
   ```bash
   npx expo start
   ```

5. O Expo abrirá uma aba no seu navegador. Você pode escanear o QR Code usando o aplicativo **Expo Go** (disponível para Android e iOS) no seu celular para visualizar o app, ou executá-lo em um emulador Android/iOS pelo seu computador.

## 📁 Estrutura de Arquivos

- `App.js`: Ponto de entrada do aplicativo com a estrutura principal da interface.
- `styles.js`: Arquivo contendo os estilos aplicados na tela.
- `package.json`: Configurações e dependências do projeto.
